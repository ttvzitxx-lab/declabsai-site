# Declabsai · Embalagens de Precisão

Site institucional da **Declabsai**, empresa especializada em embalagens personalizadas para indústria, varejo e e-commerce.

🌐 **Acesse:** [https://ttvzitxx-lab.github.io/declabsai-site/](https://ttvzitxx-lab.github.io/declabsai-site/)

---

## Sobre o projeto

Site estático desenvolvido com HTML, CSS e JavaScript puro. Design inspirado na estética de marcas de luxo — tipografia editorial, paleta preto e branco, animações refinadas.

### Seções

- **Hero** — apresentação com headline tipográfico e estatísticas
- **Portfólio** — linha completa de produtos em tabela editorial
- **Identidade** — diferenciais e números da empresa
- **Processo** — 4 etapas do pedido à entrega
- **Contato** — formulário e canais de atendimento

### Tecnologias

- HTML5 / CSS3 / JavaScript (sem dependências externas)
- Fontes: [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) + [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- Hospedagem: GitHub Pages com HTTPS automático

---

## Como atualizar

```bash
# Edite o index.html, depois:
git add index.html
git commit -m "descrição da atualização"
git push
```

O GitHub Pages publica automaticamente em 1–2 minutos após o push.
